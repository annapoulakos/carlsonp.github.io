### About

The personal website for Patrick Carlson.

### Setup and Build Instructions

Built using Jekyll.

* Git clone
* Run `bundle install` to install the dependencies via Gem
* Run `bundle exec jekyll serve --watch`
* View the site at `http://localhost:4000/`
